🤖 AI-Powered Document Summarizer & Q&A Tool
This is an AI-powered web application that allows users to upload PDF or DOCX documents, receive automatic summaries, and ask questions about the document using natural language. 
It is built using Streamlit, OpenAI GPT-4, and Python.

🚀 Features
| Feature              | Description                                                                 |
|----------------------|---------------------------------------------------------------------------- |
| 📤 Document Upload   | Upload `.pdf` or `.docx` files                                             |
| ✨ Smart Summarizer  | Extracts concise, readable summaries from long documents                   |
| 💬 Q&A Chat          | Ask questions like “What is the main topic?”, “What are the key points?”   |
| 🧠 GPT-4 Powered     | Uses OpenAI GPT-4 API for summarization and question-answering             |
| 📥 Export Summary    | Download the generated summary for future use                              |
| ☁️ Deploy Anywhere   | Easy deployment via Streamlit Cloud or Hugging Face Spaces                 |

🛠 Tech Stack
- Python 3.x
- Streamlit** – for UI
- OpenAI GPT-4 API – for summarization and question answering
- PyPDF2 / pdfplumber / docx2txt – for document text extraction
- dotenv – for managing API keys securely
- FPDF – for exporting results (optional)
